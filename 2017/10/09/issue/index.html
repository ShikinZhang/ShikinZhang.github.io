Installing AVR STUDIO 4 issue

Oct 9, 2017 | Hits

ISSUE

从LMS上下载的AVR STUDIO 会遇到一个问题，点击到installing的时候，会最小化然后就安装不了。

SOLUTION

进入安全模式下安装。

WIN10 进入安全模式

按 4 或者 F4 键进入安全模式，然后安装AVR STUIDO 4.19 即可
